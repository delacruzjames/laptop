# laptop

This sets up your Mac for development incl [dotfiles](https://github.com/kriskhaira/dotfiles).


## Instructions

1. Clone this repo somewhere.

  ```
  git clone git@github.com:kriskhaira/laptop.git ~/code/laptop
  ```
  
  
1. Run the script.
 
  ```
  ~/code/laptop/mac
  ```

1. Install the Vim plugins.

  ```
  vim
  :PlugInstall
  ```
